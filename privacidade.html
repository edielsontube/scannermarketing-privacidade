Política de Privacidade – Extensão Scanner Marketing

Esta política de privacidade descreve como a extensão Scanner Marketing lida com os dados do usuário. A extensão foi desenvolvida com foco em segurança, privacidade e transparência.

1. Coleta de Dados

A extensão não coleta dados pessoais como nome, e-mail, localização, senhas ou qualquer informação sensível do usuário.

As únicas informações acessadas pela extensão são relacionadas à página ativa do navegador, como o conteúdo visível (HTML, scripts e metadados) e a atividade na aba ativa, sempre iniciadas por ação do próprio usuário.

2. Uso de Dados

Os dados coletados localmente são usados exclusivamente para exibir ao usuário informações técnicas da página analisada. Nenhum dado é enviado para servidores externos.

A extensão também utiliza o Firebase Realtime Database para validar uma chave de ativação previamente adquirida pelo usuário. Essa validação é feita de forma segura e não envolve dados sensíveis.

3. Armazenamento

A chave de ativação é armazenada localmente no navegador para manter a licença ativa enquanto válida. Nenhuma outra informação é armazenada ou compartilhada.

4. Compartilhamento de Dados

Não compartilhamos, vendemos ou transferimos qualquer informação do usuário para terceiros, em nenhuma circunstância.

5. Código da Extensão

Todo o código da extensão é executado localmente. Não utilizamos código remoto, CDN externa ou serviços de rastreamento de terceiros.

6. Segurança

A extensão inclui mecanismos de bloqueio para evitar a execução em URLs sensíveis, como chrome://, file://, about: e outras páginas protegidas do navegador.
